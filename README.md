# Projeto Integrador IEEP - Sistema de Solicitação de Documentos Digitais

Bem-vindo ao repositório oficial do Sistema Integrado de Pedidos Escolares (SIPE). 
Este projeto visa digitalizar e otimizar a solicitação de documentos para alunos e servidores.

## Tecnologias Utilizadas

**Frontend:**
* React.js
* Tailwind CSS
* Font Awesome

**Backend & Dados:**
* Node.js
* PostgreSQL
* Autenticação via Firebase/Supabase

## Estrutura do Projeto
* `/projeto-escrito` - Documentação oficial e relatórios.
* `/site-institucional` - Landing page de apresentação do sistema.
* `/sistema-sipe` - Código-fonte da aplicação (Frontend e Backend).
